    Политика конфиденциальности ОФЭРНиО    


Для участия в регистрации электронных ресурсов авторы-индивидуалы/авторские коллективы предоставляют контактную (персональную) информацию автора/доверенного лица авторского коллектива, необходимую для присвоения регистрируемой разработке кода по ЕСПД.

Все участники процесса регистрации в ОФЭРНиО имеют право:

·         просматривать  информационные ресурсы и коллекции ОФЭРНиО без создания учётной записи;

·         использовать информационные ресурсы и коллекции ОФЭРНиО (мониторинг, статистика) без создания учётной записи (при этом ссылка на ресурсы ОФЭРНиО обязательна).

Чтобы улучшить процесс регистрации в ОФЭРНиО, собирается информация  об адресах электронной почты, когда пользователи ОФЭРНиО:

·         отправляют в ОФЭРНиО электронные письма;

·         учавствуют в обратной связм с ОФЭРНиО

ОФЭРНиО обязуется:

·         использовать все  доступные меры, чтобы сохранить персональную информацию;

·         не передавать третьим лицам персональную информацию;

·         не раскрывать третьим лицам персональную информацию;

Пользователи ОФЭРНиО, обращаясь в ОФЭРНиО, по умолчанию соглашаются с политикой конфиденциальности ОФЭРНиО, в том числе: с размещением реферативных материалов о разработке и полнотекстовых рекламно-технических описаниях на портале ОФЭРНиО.
