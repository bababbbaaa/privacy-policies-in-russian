ПРОСМОТР ВЕБ-САЙТА

Каждый раз, когда вы посещаете и просматриваете данный веб-сайт, вы делаете это конфиденциально,  исключая случаи, если вы по собственному желанию предоставляете нам эту информацию.

Если вы во время визита  не  выполняли никаких действий кроме просмотра  текста, изображений или скачивания информации без предоставления любой личной информации, данные о вашем визите автоматически регистрируется. Эти данные не идентифицируют вашу личность.

Эти данные используются в целях управления сайтом, сбором статистических данных, и в случае обнаружения  запрещённых действий, для возможности применения законодательства и судебного преследования.

Сайт автоматически регистрирует данные о посетителях для составления статистических отчётов.

Мы используем статистические данные для оптимизации сайта для посетителей, например, для подбора интересующей посетителей информации, и для других целей, таких как, определение технических спецификаций дизайна сайта и устранение проблемных частей.

Далее перечислены автоматически регистрируемые данные о посещении сайта:

- Интернет домен и IP адрес с которого вы посетили сайт

- Тип интернет браузера и операционной системы  компьютера

- Дата и время визита

- Страницы которые вы посетили

Если вы перешли с другого сайта, регистрируется  сайт с которого вы перешли на наш сайт

Если вы нашли наш сайт посредством поисковой системы, регистрируется поисковая система и ключевые слова, которые вы использовали для поиска
