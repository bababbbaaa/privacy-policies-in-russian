Политика конфиденциальности

Группа компаний «Orange Code» (далее – Orange Code) уважает право каждого человека на конфиденциальность.

Для использования некоторых служб, программ и продуктов Orange Code (далее — Продукты Orange Code) может понадобиться прохождение процедуры регистрации или предоставление персональной информации Orange Code. Регистрация может понадобиться для доступа к некоторым функциям сайтов или участия в рекламных акциях. Условия о конфиденциальности содержат пояснения, как используется такая информация. Внимательно ознакомьтесь с настоящей Политикой конфиденциальности и при необходимости задайте вопросы.

Настоящая Политика конфиденциальности регулирует отношения между Orange Code и любым физическим лицом, индивидуальным предпринимателем, юридическим лицом (далее – Пользователь) по обработке Персональной информации, включая, помимо прочего, сбор, использование, сохранение и безопасность предоставленной Пользователем и/или собранной у Пользователя информации. Действие настоящих Условий о конфиденциальности распространяются на всю информацию, которую Orange Code и/или его аффилированные или иные уполномоченные лица, действующие от имени Orange Code, могут получить о Пользователе во время использования им любого из Продуктов Orange Code.

Обращаем Ваше внимание: использование Продуктов «Orange Code» означает Ваше полное согласие со всеми положениями настоящей Политики конфиденциальности и указанными в ней условиями обработки Вашей персональной информации. Если Вы не согласны безоговорочно принять все положения настоящих Условий о конфиденциальности, Вы не имеете права использовать Продукты Orange Code и должны удалить все их компоненты со своего компьютера (ЭВМ).

1. Персональная информация, собираемая и обрабатываемая Orange Code

В рамках настоящей Политики конфиденциальности под персональной информацией понимается:

  1. Информация (включая, но не ограничиваясь, персональные данные), которую Пользователь самостоятельно предоставляет Orange Code при осуществлении действий, направленных на регистрацию Продуктов Orange Code, либо в рамках их использования (в т.ч. создание учетной записи, обращение в службу поддержки). Информация предоставляется путем заполнения Пользователем соответствующих форм, при этом объем предоставления информации при заполнении необязательных полей указанных форм определяется Пользователем самостоятельно.
  2. Информация, получаемая автоматически при использовании Продуктов Orange Code Пользователем, в т.ч. из cookies (при загрузке материалов сайтов 1С-Битрикс, автоматически проверяются сведения о системе Пользователя; при использовании инструментов общения Пользователем, могут быть сохранены материалы и сообщения, которые он отправлял и получал, при этом Orange Code оставляет за собой право проверять содержание общения Пользователя на своих сайтах).
  3. Пользовательские данные – любая информация загруженная (внесенная) Пользователем в Продукты Orange Code.
  4. Иная информация о Пользователе, которую получает Orange Code исключительно в указанных в разделе 2 настоящих Условий о конфиденциальности целях.

2. Цели сбора и обработки персональной информации пользователей

Сбор, хранение и обработка указанной в разделе 1 настоящих Условий о конфиденциальности информации осуществляется исключительно в следующих конкретных целях:

  1. Для идентификации Пользователя при использовании Продуктов Orange Code;
  2. Для персонализации учета посещений (в том числе с целью предоставления персонализированных услуг Пользователю) и регистрации действий Пользователя Orange Code использует Cookies.
  3. Для осуществления взаимодействия с Пользователем в рамках заключенных с Orange Code договоров или соглашений.
  4. Проведение статистических и иных исследований на основе обезличенных данных (в т.ч. для выяснения мнения Пользователей о различных продуктах и услугах, оказываемых Orange Code).
