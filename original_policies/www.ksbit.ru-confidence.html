Политика конфиденциальности

Компания "КСБИТ" обязуется соблюдать настоящую Политику, заботясь о защите личных данных пользователей. 

​

ЛИЧНЫЕ ДАННЫЕ

​

К личным данным пользователя сайта относится вся информация, содержащаяся в обязательных для заполнения полях форм сайта, а именно: 

  •     имя пользователя; 

  •     его электронный адрес; 

  •     название компании; 

  •     телефон; 

  •     адрес физический (в случае оформления заказа на физическое лицо); 

  •     адрес юридический (в случае оформления заказа на юридическое лицо); 

  •     данные об объёме заказываемой продукции. 

​

СБОР ЛИЧНЫХ ДАННЫХ

​

Мы ставим посетителей сайта в известность, если нам потребуется получить информацию, которая позволяет установить их личность или их контактные данные. Эти данные запрашиваются при регистрации, а также при подписке на новости, рассылаемые по электронной почте и при получении доступа к закрытым тематическим разделам сайтам.

На сайте компании "КСБИТ" осуществляется сбор статистики посещаемости сайта.

Статистическая информация может включать в себя:

  •     IP-адрес пользователя; 

  •     тип браузера; 

  •     количество предыдущих посещений; 

  •     адрес сайта, с которого пользователь осуществил переход на наш сайт. 

Данная информация используется для повышения качества предоставляемых услуг, а также для подготовки обобщенных статистических данных, характеризующих использование сайта. 

​

ИСПОЛЬЗОВАНИЕ ЛИЧНЫХ ДАННЫХ

​

Личные данные посетителей сайта необходимы компании "КСБИТ" для решения ряда задач: 

  •  предоставление спектра услуг: 

    •       о регистрации на мероприятия "КСБИТ"; 

    •       о подписке на рассылки информационных бюллетеней, пресс-релизов, отчётов; 

    •       о приглашении на мероприятия "КСБИТ"; 

    •       oб организации доступа для загрузки программного обеспечения; 

    •       oб организации доступа для загрузки документации по продуктам. 

  • оперативное оповещение об обновлениях в разделах продуктов и услуг, а также информирование об обновлениях информации на сайте, предоставляемых по запросу пользователей; 

  • предоставление ограниченного доступа к защищенным разделам сайта.

​

КОНТРОЛЬ ЛИЧНЫХ ДАННЫХ

​

Компания "КСБИТ" обязуется не использовать личные данные в каких-либо иных целях, кроме перечисленных выше. Эти данные не будут передаваться третьим организациям и лицам без разрешения пользователей, за исключением случаев, в которых этого требует действующее законодательство РФ.

​

ДОСТУП К ЛИЧНЫМ ДАННЫМ

​

Посетителям сайта предоставляются средства, которые позволяют им проверять точность указанных личных данных и производить их обновление в случае необходимости.

​

ОБЕСПЕЧЕНИЕ БЕЗОПАСНОСТИ ЛИЧНЫХ ДАННЫХ

​

"КСБИТ" уделяет значительное внимание обеспечению информационной безопасности пользователей, мы применяем технологии и процедуры, помогающие защитить их данные от кражи, модификации, возможности несанкционированного доступа, нелегального использования этих данных или их разглашения.

 

Тем не менее, Интернет не безопасен, и компания "КСБИТ" не может гарантировать безопасную передачу любой информации по открытым каналам.

​

ИСПОЛЬЗОВАНИЕ ФАЙЛОВ "cookie"

​

Файлы "cookie" предназначены для сбора сведений о том, какие ссылки на сайте пользуются наибольшей популярностью у посетителей и используются для повышения эффективности работы корпоративного ресурса.

 

Сбор этих данных осуществляется в обобщенном виде и никогда не соотносится с личными сведениями пользователей.

 

Пользователь может отключить в браузере прием файлов "cookie" с сайта компании "КСБИТ", при этом он сможет просматривать текст на веб-страницах, однако возможности индивидуальной настройки на службы сайта будут недоступны.

​
