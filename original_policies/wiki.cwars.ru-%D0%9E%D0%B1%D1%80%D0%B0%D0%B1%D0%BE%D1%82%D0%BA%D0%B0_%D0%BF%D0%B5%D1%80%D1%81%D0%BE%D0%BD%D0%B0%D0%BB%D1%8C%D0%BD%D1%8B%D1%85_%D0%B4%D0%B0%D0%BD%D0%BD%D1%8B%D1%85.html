Обработка персональных данных

Материал из Игры ClanWars - браузерная игра из серии игр: Бойцовский клуб, Комбатс, Олд БК

1.1. Пользователь дает согласие Администрации на сбор и обработку своих персональных данных с целью обеспечения реализации правоотношений между Пользователем и Администрацией, которыми регулируются условия пользования Сайтом. Пользователь также дает согласие Администрации на передачу своих персональных данных третьим лицам в случаях, предусмотренных законодательством Украины и Соглашением об использовании сервиса www.cwars.ru и Правилами о защите информации.

1.2. Администрация гарантирует, что все ее сотрудники, которые допущены к любому виду обработки персональных данных Пользователя, будут строго придерживаться требований законодательства Украины по охране персональных данных.

1.3. Персональные данные Пользователя, обрабатываемые Администрацией, состоят из данных, которые Пользователь передал Администрации для регистрации на Сайте (дата рождения, электронный адрес) и в других случаях, предусмотренных в Соглашении об использовании сервиса www.cwars.ru, а также из любого вида данных, которые Пользователь на свое усмотрение опубликовал на Сайте, как для всеобщего, так и для ограниченного ознакомления и/или сообщил ссылку, по которой с такими данными можно ознакомиться.

1.4. Для сбора статистических данных и идентификации посетителей Сайта Администрация может отслеживать и сохранять информацию об IP-адресах доступа Пользователя к Сайту, использовать файлы технической информации (cookies), размещаемые на локальном терминале Пользователя или посетителя Сайта.

1.5. Администрацией также обрабатываются данные Пользователя, которые автоматически получаются сервером во время доступа Пользователя к Сайту и дальнейших действий Пользователя на Сайте.

1.6. Администрацией также обрабатываются данные Пользователя, которые получены в результате действий других пользователей Сайта.

1.7. Пользователь соглашается с тем, что обязательные данные, предусмотренные п. 4.1 в Соглашении об использовании сервиса www.cwars.ru, а также данные, которые Пользователь на свое усмотрение опубликовал на Сайте, данные Пользователя, которые получены в результате действий других пользователей Сайта, и которые носят конфиденциальный характер, могут собираться, храниться, использоваться и распространяться Администрацией, а потому не будет нарушением прав Пользователя на защиту персональных данных и на конфиденциальность такой информации со стороны Администрации размещение на Сайте таких данных для ознакомления третьих лиц.

1.8. Полученные данные используются для:

- электронный адрес - коммуникация с Пользователем; информирование про события проекта; безопасность (восстановление и смена паролей);

- дата рождения - идентификация возраста Пользователя; безопасность (восстановления доступа, если утерян доступ к электронному адресу);

- все остальные данные используются, только для обеспечения всех возможностей игрового процесса Пользователя на Сайте.

1.9. Удаление данных:

- Пользователь может обратиться к Администрации Сайта и запросить блокировку своей учетной записи;

- через 7 месяцев, с момента блокировки (если Пользователь больше не использовал свою учетную запись), все данные будут удалены автоматически.

2.0. Запрет на обработку персональных данных автоматически влечет за собой прекращение действия настоящего договора и отмену статуса Пользователя с момента такого запрета.

2.1. Администрация без дополнительного разрешения Пользователя может передавать персональные данные такого Пользователя третьим лицам, в т.ч. государственным органам, только в случаях и на основаниях, прямо предусмотренных действующими законами Украины, когда такая передача становится обязанностью Администрации.

2.2. Администрация обязуется обеспечить сохранность персональных данных Пользователя от случайной потери или уничтожения, от незаконной обработки, в т.ч. незаконного уничтожения или доступа к персональным данным.
