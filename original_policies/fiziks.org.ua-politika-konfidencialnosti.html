Сбор, цель использования и порядок обработки персональных данных

К персональным данным относят сведения о физическом лице, прошедшие идентификацию или который можно идентифицировать.

Для обработки персональных данных используются серверы Компании. Сбор персональных данных производится для облегчения последующего пользования Сайтом. Будут учтены интересы, предпочтения пользователя при осуществлении редакторской политики и работе с контрагентами в сфере рекламы.

Продолжительность хранения персональных данных не превышает срок, необходимый для их обработки.

Порядок использования файлов Сookie

Файл Cookie является маленьким текстовым файлом, сохраняемым Сайтом на вашем мобильном телефоне или компьютере, при его посещении. Считать этот файл может только сам Сайт. Постоянные файлы cookie остаются на компьютере после закрытия браузера. Сессионные удаляются после закрытия браузера. Потенциально возможно использование сторонних сервисов, устанавливающих свои файлы cookie, именуемые файлами cookie сторонних партнеров. При посещении сайта вы можете принять либо отклонить файлы cookie.

Нами могут использоваться «веб маяки» или схожие технологии. Они помогают запомнить имя пользователя, язык и другие сведения на некоторый временной интервал, чтобы вы не вводили эти данные на каждой странице при посещении сайта. С их помощью мы оцениваем
