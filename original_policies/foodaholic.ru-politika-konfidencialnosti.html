Политика конфиденциальности

На данной странице содержится информация, которую мы получаем и собираем, когда вы пользуетесь сайтом https://foodaholic.ru/ (далее foodaholic.ru или Администрация). Мы обязуемся сохранять вашу конфиденциальность в сети Интернет. Настоящая Политика Конфиденциальности, рассказывает о том, как собираются, обрабатываются и хранятся ваши личные данные. Администрация уделяет большое внимание защите личной информации пользователей. Пользуясь сайтом foodaholic.ru, пользователь тем самым дает согласие на применение правил сбора и использования данных, изложенных в настоящем документе.

Если Вы не согласны с условиями нашей политики конфиденциальности, не используйте сайт foodaholic.ru!

Собираемая информация

Администрация сайта может собирать следующую информацию о пользователях сайта:

  • Имя
  • Адрес электронной почты
  • IP-адрес
  • Прочая информация

Использование информации

Ниже описаны некоторые способы использования личной информации пользователя:

  • для внутреннего отчета
  • для предоставления информации и услуг, которые запрашивает пользователь
  • для ответа на запросы пользователя
  • для улучшения качества продуктов сайта
  • для улучшения качества предоставления услуг
  • для мониторинга работы сайта
  • для отправки различной электронной корреспонденции
  • для проведения опросов
  • для формирования статистических данных

Во время подписки на рассылку сайта пользователь предоставляет адрес электронной почты и имя.

Раскрытие информации

Администрация не продает личные данные пользователя и не передает их третьим лицам без согласия на то пользователя, за исключением случаев предусмотренных законодательством.

Использование файлов cookie

  • Сайт foodaholic.ru использует файлы cookie для улучшения работы сайта, в частности для улучшения отображения сайта в используемом вами браузере.
  • Некоторые сторонние рекламные системы, используемые на сайте, также могут использовать файлы cookie для показа рекламы. О том, как рекламная система goolge adsense использует файлы cookie вашего компьютера, вы можете прочитать здесь — https://www.google.com/policies/technologies/ads/
  • Если вы не желаете, чтобы данный сайт и иные сайты вообще не использовали файлы cookie вашего компьютера, вы можете заблокировать их использование в браузере.
