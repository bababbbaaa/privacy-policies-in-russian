1.1.9. “IP-адрес” – уникальный сетевой адрес узла в компьютерной сети, построенной по протоколу IP.
1.1.10. “Товар” – вещь, услуга (работа), которую Пользователь приобретает (покупает) с помощью сайта.
1.1.11. “Блокирование персональных данных” – временное прекращение обработки персональных данных (за исключением случаев, если обработка необходима для уточнения персональных данных).
1.1.12. “Распространение персональных данных” – действия, направленные на раскрытие персональных данных неопределенному кругу лиц.
