   Создание системы защиты персональных данных в соответствии с положениями Федерального закона «О персональных данных» и требованиями нормативных документов по защите персональных данных позволяет минимизировать правовые и репутационные риски, связанные с несоблюдением законодательства России в области автоматизированной обработки персональных данных.



 Secret Net Studio

    Комплексное решение для защиты рабочих станций и серверов на уровне данных, приложений, сети, операционной системы и периферийного оборудования

 

Dr.Web Enterprise Security Suite 10.0

    Централизованная защита всех узлов корпоративной сети
 

RedCheck

    Программный комплекс для анализа защищенности и управления информационной безопасностью.
 

Стек-НС1в

    Устройство экстренного уничтожения информации.
 

Microsoft Office Professional Plus 2013

    Программное обеспечение, включающее в себя клиентские и серверные приложения.
 

Microsoft Hyper-V Server 2012

    Операционная система с единственной ролью - сервером виртуализации.
 

Kaspersky Security Center 10

    Инструмент для централизованного управления комплексной системой защиты, который предоставляет администратору доступ к детальной информации об уровне безопасности корпоративной сети и позволяет гибко настраивать все компоненты системы защиты.

 

CSP VPN Gate. Версия 3.11

    Программный комплекс, предназначенный для защиты передаваемой по протоколу TCP/IP открытой и конфиденциальной информации, не содержащей сведений, составляющих государственную тайну, в информационных системах.
 

CSP VPN Server. Версия 3.11

    Программный комплекс предназначенный для создания защищенных соединений между сервером VPN и другими взаимодействующими с ним доверенными шлюзами VPN и/или клиентами VPN, а также может выполнять роль межсетевого экрана.
 

Security Studio Endpoint Protection

    Средство защиты информации.
 

Аккорд-Х

    Программно-аппаратный комплекс средств защиты информации (ПАК СЗИ) предназначенный для разграничения доступа пользователей к рабочим станциям под управлением ОС семейства Linux.
 

РОСА DX «ХРОМ» 1.0

    Операционная система для настольных систем  со встроенными средствами защиты от несанкционированного доступа к информации.
 

Cisco 2900

    Маршрутизаторы серии Cisco 2900
(модели Cisco 2901, Cisco 2911, Cisco 2921 с установленным программным обеспечением Cisco IOS 15.0(1)M5/Cisco IOS 15.2(4)M4, модель Cisco 2951 с установленным программным обеспечением Cisco IOS 15.0(1)M2).
 

РЧ СПО Навигатор-С

    Расчетная часть специального программного обеспечения «Навигатор-С»
 

Kaspersky Endpoint Security 10 для Windows

    Антивирусное средство защиты корпоративных пользователей от информационных угроз.
