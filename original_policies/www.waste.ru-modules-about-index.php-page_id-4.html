Политика конфиденциальности

1. Общие положения

1.1 Настоящий документ «Политика конфиденциальности» (далее – «Политика») представляет собой правила использования сайтом «Отхорды.Ру» waste.ru (далее «сайт») персональной информации Пользователя (посетителя сайта). И действует в отношении всей информации, которую сайт, может получить о посетителе.

1.2 Использование вышеуказанного сайта означает безоговорочное согласие Пользователя с настоящей Политикой и указанными в ней условиями обработки его персональной информации. В случае несогласия с этими условиями Пользователь должен прекратить использование сайта.

1.3 Настоящая Политика конфиденциальности применяется только к сайту «Отхорды.Ру». Сайт не контролирует и не несет ответственность за сайты третьих лиц, на которые Пользователь может перейти по ссылкам, доступным на сайте.

2. Персональная информация

2.1 Под персональной информацией в настоящей Политике понимается:
- Информация, которую Пользователь предоставляет о себе самостоятельно при использовании сайта, включая персональные данные Пользователя, например при заполнении формы обратной связи.

- Данные, которые передаются в автоматическом режиме в зависимости от настроек программного обеспечения Пользователя, включая, но не ограничиваясь: IP-адрес, cookie, данные об используемом Пользователем программном обеспечении и оборудовании для работы в сети связи, включая Интернет, параметрах и настройках интернет-браузеров, передаваемой и получаемой с использованием Сервиса информации и материалах.

2.2 Пользователь осознает и принимает возможность размещения на страницах сайта программного обеспечения третьих лиц, к которому относятся:
— системы по сбору статистики посещений
— социальные плагины (блоки) социальных сетей

3. Цели сбора Персональной информации пользователя

3.1 Персональная информация пользователя может использоваться с целью:
- Осуществления обратной связи с пользователем, предоставления эффективной клиентской поддержки.

4. Отказ от ответственности
4.1 Сайт не осуществляет проверку достоверности предоставляемой Персональной информации.

5. Защита Персональной информации

5.1 Сайт не осуществляет хранение Персональной информации. Сайт не содержит базу данных пользователей и не осуществляет рассылку электронной почты. Для доступа к информации расположенной на сайте Пользователю не нужно указывать свои персональные данные.

5.2 Владелец сайта вправе передавать полученную информацию о посетителе третьим лицам по запросу суда или иного уполномоченного государственного органа в рамках установленной законодательством процедуры.

6. Заключение
6.1 Владелец сайта вносит изменения в настоящую Политику конфиденциальности без согласия Пользователя. Новая Политика конфиденциальности вступает в силу с момента ее размещения на сайте, если иное не предусмотрено новой редакцией Политики конфиденциальности.

6.2 Все предложения, претензии, отзывы или вопросы по настоящей Политике Конфиденциальности следует сообщать на адрес электронной почты info@waste.ru
