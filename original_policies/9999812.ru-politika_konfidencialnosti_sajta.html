Какая информация сохраняется на нашем сайте

Автоматически сохраняются программным обеспечением сайта в журналах сервера техническая информация, поступающая в момент посещения сайта:

  • тип браузера и ОС;
  • данные сайта, с которого был осуществлен переход;
  • время запроса.

Техническая информация собирается счетчиком статистики. Может использоваться для обезличенного анализа посещений, разработки методов улучшения работы сайта, компании.

Дополнительная информация собирается из предоставленных данных в любой заполненной вами форме. Объем личных данных определяет сам пользователь. Эти сведения могут быть использованы для обратной связи или для выполнения расчетов, оказания заказанных услуг. Также информация может быть применена для расширения услуг по индивидуальным запросам, улучшения системы поддержки пользователей.

Основные положения конфиденциальности

Использование файлов cookies

Небольшой фрагмент данных на текстовом файле (cookie) хранятся на веб-браузере пользователя и отправляются на наш веб-сервер с запросом. Сokies не несут личной информации, но улучшают пользователю доступ к сайту, определяет повторные посещения. На нашем сайте они используются для ведения статистики и как технология идентификации пользователей при повторном посещении.

Свое участие в применении технологии (cookie) можно ограничить,введя в настройки вашего браузера соответствующие изменения.
