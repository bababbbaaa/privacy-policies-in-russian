Политика конфиденциальности

Администрация сайта UNI-WEAR.RU (далее Администрация) обязуется сохранять Вашу конфиденциальность в сети Интернет. 

Настоящая Политика Конфиденциальности, рассказывает о том, как собираются, обрабатываются и хранятся Ваши личные данные. Администрация уделяет большое внимание защите личной информации пользователей. Пользуясь сайтом UNI-WEAR.RU, пользователь дает согласие на применение правил сбора и использования данных, изложенных в настоящей политике. 

Все контактные формы на сайте сделаны для Вашего удобства и служат для скорейшей связи с менеджерами компании и отправки запросов и заказов.

Собираемая информация (Персональные данные)

Администрация сайта может собирать следующую информацию о пользователях сайта:
— Фамилия, Имя, (Отчество)
— Адрес электронной почты
— Номер телефона

— Место жительства Покупателя, адрес доставки готовой продукции заказчику
— Прочая информация

Использование информации (Обработка персональных данных)

Ниже описаны некоторые способы использования личной информации пользователя:
— для предоставления информации по заказу и услуг, которые запрашивает пользователь
— для ответа на запросы пользователя и связи по электронной почте и уточнения деталей заказа по телефону, для грамотного обращения к клиенту
— для улучшения качества продуктов сайта
— для улучшения качества предоставления услуг
— для мониторинга работы сайта
— для отправки различной электронной корреспонденции, связанной с услугами компании
— для формирования статистических данных внутри компании

— для внутренней отчетности

- Для передачи третьим лицам, в частности, курьерским службам, организациями почтовой связи, операторам электросвязи, исключительно в целях выполнения заказа Пользователя, оформленного на Сайте https://uni-wear.ru, включая доставку Товара.
