Материал из CrimLib.info

Сайт Crimlib.info создан для распространения проверенных сведений, касающихся криминалистики и уголовного процесса.

Для просмотра содержимого сайта не требуется регистрация, абсолютно все материалы сайта доступны в открытом доступе. Регистрация может потребоваться исключительно для получения возможности внесения изменения в материалы сайта. При создании учётной записи пользователя мы просим указать только имя пользователя и пароль, с помощью которых пользователь сможет пользоваться сайтом. Указание адреса электронной почты не является при регистрации обязательным, адрес электронной почты может быть использован только для восстановления имени пользователя и пароля.

Сайт Crimlib.info не осуществляет запись и хранение персональной информации и не является оператором персональных данных в соответствии с ФЗ №152 "О персональных данных".

О допущенных нарушениях просьба сообщить администрации сайта письмом на электронную почту ae @ crimlib.info (без пробелов).
