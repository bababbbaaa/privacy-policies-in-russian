Файлы cookie
Файл cookie содержит личные данные пользователя в зашифрованном виде, а также другую персональную информацию, включая настройки пользователя на сайте. Ранее полученные файлы cookie можно удалить. Инструкции по удалению cookie включены в соглашение о конфиденциальности вашего браузера, в частности, для Microsoft Internet Explorer, Mozilla Firefox и Google Chrome. При удалении файлов cookie все настройки и параметры, которыми руководили ими, будут удалены.

Сайт RIDER используют файлы cookie для следующих целей:
Аналитика: RIDER может использовать cookie для подсчета количества уникальных посетителей веб-страницы или онлайн-сервиса, учитывать эти данные в статистике и аналитике. Эти данные помогают улучшить работу сайтов и сервисов.
Хранение персональных настроек: сайт RIDER может хранить в файлах cookie данные о просмотренных товары и корзину, персональную информацию пользователя (например, имя, адрес электронной почты, город, пункт выдачи перевозчика), которые были введены в процессе заполнения электронной формы для отображения соответствующие локальные настройки при последующих посещениях и не требовать повторного ввода этих данных.
