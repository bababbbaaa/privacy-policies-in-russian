Файлы cookie

Файл cookie представляют собой небольшой фрагмент данных, отправляемый с веб-сайта и сохраняемый в интернет-браузере пользователя, когда пользователь просматривает веб-сайт. Позже при просмотре веб-сайта, пользователь может извлечь данные, сохраненные в файле cookie.
Файлы cookie были разработаны как надежный механизм, с помощью которого интернет-сайты запоминают свое состояние или действия, которые пользователь выполнял ранее. В них могут сохраняться информация сочетаниях клавиш, входе в систему, список страниц, посещенных пользователем.
Существуют файлы cookie, которые выполняют функции аутентификации. Это наиболее распространенный метод для защиты доступа к системе нежелательных пользователей. Один раз войдя в систему Вы можете постоянно оставаться залогиненным в системе. Без такого механизма веб-сайт непрерывно бы требовал аутентификации при отправке конфиденциальной информации.

Типы cookie которые мы используем

На веб-сайте balans.cz используются следующие типы файлов cookie:
а) сессионные cookie — находящиеся во временной памяти на период посещения веб сайта. После окончания сессии, cookie автоматически удаляются.
б) cookie производительности — позволяют измерять и улучшать производительность сайта.
в) строго необходимые cookie — используются для работы с сайтом, вход в систему, переход между страницами, заполнение форм.
г) сторонние cookie для Google Analytics — помогают узнать какие страницы более популярные, какие менее, откуда пришли пользователи веб сайта. Эта информация используется для улучшения качества наших услуг.

Управление cookie
При нежелании, чтобы Ваш браузер принимал, можно отключить опцию принятия cookie в настройках Вашего браузера.
Для получения дополнительной информации о том, как отключить cookie, см. вкладку «Справка» Вашего браузера.
