Безопасность

Наш веб-сайт RtiIvaz.ru использует специализированное программное обеспечение для обеспечения безопасности и защиты сайта Shield Security. Это помогает гарантировать, что утечки данных не происходят, а наш веб-сайт и данные защищены от попыток взлома и вторжения.

Безопасность Shield защищает посетителей сайта и блокирует потенциальные взломы, отслеживая веб-трафик и изменения файловой системы.

Чтобы узнать больше о безопасности Shield, пожалуйста, перейдите по этой ссылке.

Cookies

Плагин Shield Security никогда и ни в коем случае не хранит какую-либо конфиденциальную личную информацию в файлах cookie.

В случае, если плагину безопасности Shield необходимо перенаправить посетителя или любой запрос, он может использовать файл cookie для предотвращения повторяющихся / бесконечных циклов перенаправления.

Для зарегистрированных /вошедших в систему пользователей плагин Shield Security использует файл cookie для отслеживания пользовательских сеансов и управления отображением определенных уведомлений администратора в плагине.

Плагин Shield Security обычно не использует файлы cookie для незарегистрированных посетителей сайта. Однако он может использовать файл cookie для регистрации закрытия значка безопасности Shield Security, чтобы предотвратить повторное отображение.

Хранение данных: Пользовательские Сеансы

Для вошедших в систему пользователей плагин Shield Security хранит информацию об имени пользователя, IP-адресе и времени последнего входа в систему и последнего действия.

Эта информация удаляется при выходе из системы или очистке данных.

Хранение данных: Контрольный журнал

Плагин Shield Security имеет функцию журнала аудита, которая будет регистрировать следующую информацию:

  • Имя пользователя, вошедшего в систему (где применимо);
  • Исходный IP-адрес запроса.

Для зарегистрированных пользователей это представляет собой информацию, которая может быть использована для определения местоположения (по IP-адресу) и идентификации отдельных лиц и их активности на сайте.
