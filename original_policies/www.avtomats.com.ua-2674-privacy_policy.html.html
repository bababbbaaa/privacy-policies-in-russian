      Политика конфиденциальности интернет-магазина «Легор»

      Использование Сайта и сотрудничество с целью поставки обозначает безоговорочное согласие с Политикой, размещённой на этой странице.
       

      1. Личная информация Пользователя и Заказчика, которую получает Легор:

      • данные, которые автоматически передаются на счётчики посещений сайта (Яндекс Метрика, Google Analytics, Rambler TOP 100, Electro TOP): IP-адрес, данные из cookie, информация о браузере, время доступа, URL запрашиваемой интернет-страницы, длительность сессии и другая информация;
      • данные, которые передаются при помощи формы заказа конкретного товара (имя, телефон и электронная почта);
      • данные получаемые в телефонном режиме для отправки груза (ФИО, телефон, город получателя);
      • данные получаемые по факсу или электронной почте (копии свидетельства о регистрации предприятия, свидетельства плательщика НДС, доверенности) для выписки документов, сопровождающих товар.
      Данная Политика распространяется только в сфере доступности Легор, и не действует:
      • на сайтах третьих лиц (владельцы интернет-проектов, которые имеют программное обеспечение для учёта посещений Сайта);
      • на сайтах третьих лиц, на которые переходит Пользователь по ссылкам, расположенным на страницах Сайта.
      Легор не проверяет подлинность получаемой информации и возлагает на Пользователя функции для поддержки её актуальности и действительности.
       

      2. Цели собирания информации

      Легор сохраняет и обрабатывает только те персональные данные, которые нужны:
      • для отправки товара Заказчику;
      • для выписки документов сопровождающих товар;
      • для осуществления сервиса, предоставляемого Заказчикам;
      • для улучшения качества коммерческой деятельности;
      • для проведения маркетинговых исследований с предварительным обезличиванием данных.
      Легор не собирает персональные данные:
      • для рассылки рекламных материалов собственного и постороннего производства;
      • для продажи их третьим лицам в виде базы коммерческих данных.

      3. Условия обработки персональных данных Заказчика и передачи их третьим лицам

      Легор имеет право передать личные данные Заказчика третьим лицам если:
      • Заказчик дал соответствующее разрешение;
      • для предоставления сервиса необходимы такие действия (отправка товара транспортно-экспедиционной службой);
      • передача предусматривается украинским законодательством с выполнением всех указанных в законе процедур;
      • в целях защиты собственных интересов и прав при не выполнении Заказчиком своих обязательств (отказ от оплаты за отгруженный товар).
      Легор руководствуется рамками «Закона України про захист персональних даних».
       

      4. Изменение Заказчиком своих персональных данных

      Заказчик имеет право в любой момент времени внести изменения в хранящиеся персональные данные либо направить письменное уведомление о полном удалении персональных данных.
      Удаление произойдёт за исключением данных, необходимых для отчётности перед Министерством налогов и сборов Украины.
       

      5. Меры, применяемые для защиты персональных данных

      Легор применяет необходимые технические и физические меры для защиты персональных данных Пользователей и Заказчиков от случайного либо умышленного доступа к ним посторонними лицами.
       

      6. Изменения, вносимые в политику конфиденциальности

      При внесении изменений в политику конфиденциальности предыдущая версия отправляется в архив (указывается ссылка), а на данной странице размещается обновлённая Политика. Новая редакция вступает в силу с момента её публикации.
