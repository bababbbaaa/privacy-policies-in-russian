Политика конфиденциальности приложений

Персональная информация - любая информация, которая позволяет идентифицировать человека, к которому она относится.

Мобильное устройство - смартфон iPhone или планшет iPad с операционной системой iOS или iPadOS.

Обработка персональных данных может отличаться в различных программных приложениях:

 

Для приложения QTHnow:

Никакие персональные данные не запрашиваются и не сохраняются в приложении QTHnow. Данное приложение использует готовые API сайта (сервера) tnxqso.com для отправки запросов, содержащих в себе координаты (широту и долготу) местоположения мобильного устройства, отправка данных осуществляется с частотой в 10 или 60 секунд (в зависимости от выбора пользователя). QTHnow получает в ответ следующие данные: RDA, RAFA, код Locator и код QTH field - если эти данные имеются на сервере для текущего местоположения пользователя/мобильного устройства.

Активность пользователей приложения QTHnow и их координаты обрабатываются только на стороне сервера tnxqso.com в соответствии с его настройками - подробности этой обработки и хранения этих данных вы должны уточнять у администраторов ресурса tnxqso.com и контролировать изменения самостоятельно.

В приложении QTHnow на вкладке "Настройки" (Settings) вы можете авторизоваться, используя свою учетную запись сайта tnxqso.com для отправки на сервер полных запросов, включающих в себя токен учетной записи.

Приложение QTHnow запрашивает доступ к фотогалерее мобильного устройства - только для сохранения скриншотов (вкладка QTH). Приложение QTHnow запрашивает доступ к службе геолокации мобильного устройства для получения координат.

Подробное описание работы программы QTHnow здесь: https://tnxqso.com/#/qthnow

 

Для приложения Photo Deleter:

Персональные данные не запрашиваются у пользователей, не сохраняются ни в каком виде в приложении. Регистрация не требуется. Активность и геолокация пользователей не фиксируется. Запрашивается доступ к фотографиям и видео (стандартное приложение Фото) - только для выполнения функции приложения - удалять фото и видео (перемещает в папку "Недавно удаленные").

 

Для приложений Flightplan+, CreditCalc, PM-Admin и ECL An-148:

Персональные данные не запрашиваются у пользователей, не сохраняются ни в каком виде в приложениях. Регистрация не требуется. Активность и геолокация пользователей не фиксируется.

 

Для приложения Flight Document System (FDS):

Следующие персональные данные требуются для выполнения активации приложения:

- действующий email.

Следующие персональные данные могут быть сохранены на сервере, если пользователь при активации планшета укажет их:

- Имя и/или фамилия пользователя;
- номер телефона;

Приложение FDS после установки запрашивает разрешение на доступ к геолокации пользователя. Данные по геолокация сохраняются на сервере только при выполнении синхронизации. Вся информация доступна для просмотра администратором EFB (пользователем с административными правами).

Передача данных СЕРВЕР - ПРИЛОЖЕНИЕ - СЕРВЕР выполняется только через защищенное соединение (SSL).

Никакие персональные данные, временно хранимые на сервере, не передаются третьим лицам и не используются для e-mail- и СМС-рассылок. Данные по геолокации никак не анализируются и никуда не транслируются.

После удаления пользователя администратором EFB все указанные выше персональные данные удаляются с сервера.

Резервные копии данных сервера хранятся в течение 1 года, после чего автоматически удаляются. Частичное извлечение данных из резервных копий не производится.
